# LogLens - Simplified Security Log Analysis

![LogLens Banner](screenshots/dashboard-top.png)

## Overview

LogLens is a lightweight Security Information and Event Management (SIEM) tool designed to help system administrators analyze web server logs and identify potential security threats.

Web server logs contain valuable evidence about malicious activity, but analyzing thousands or millions of log entries manually is extremely difficult. LogLens automates this process by parsing Apache/Nginx logs, detecting known attack patterns, enriching events with intelligence data, and presenting the results through an interactive security dashboard.
